# README

## サービス概要

本のなかの一文を投稿するサイト。トップ画面では投稿された一文の中からランダムで表示する。立ち止まって読んでもらいたいので、縦書きを採用する。引用文を選択すると書誌情報、販売ページへと飛ぶことができる。

## このサービスへの思い・作りたい理由

- 紀伊國屋書店の「ほんのまくらフェア」で「ほんのまくら」＝「本の出だし」の一文のみの情報から本と出会うという体験が心に残っている。
- 同時に「本の出だし」という機械的に抽出可能な一文よりも、もっと個人的に痺れた/惹かれた/参った/打ちのめされた一文が閲覧可能なWebアプリがほしいと考えた。

## ユーザー層について

本好き/活字好き
次に読む本を探している人

おもしろい・興味深い一文に出会いたいと思っている層を想定しています。立ち止まって一文を読んでもらいたいので縦書きのレイアウトを採用します。

## サービスの利用イメージ

### ユーザー操作
- 書誌登録、本の一文を投稿、SNSに共有、販売ページに遷移

### 表示と共有
- 投稿された一文はトップ画面にランダムに表示
- トップ画面は「さらに表示」付きの無限スクロール  
- 一文をクリックすると書誌情報、通販サイト・SNSへの共有ボタンが表示される
  
### どんな価値が得られるか

本屋で立ち読みをする感覚で、気軽に次に読む本を選ぶことができる。

## ユーザーの獲得について
- SNS共有ボタンをつける

## サービスの差別化ポイント・推しポイント

### 類似サービス
「本の書き出し」というサービスがあります。

### 差別化ポイント
- 上記サイトの投稿は終了している模様
- 本の「書き出し」だけでなく、個人の選別した一文を掲載する
